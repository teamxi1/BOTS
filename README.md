## 📋 Sobre el Bot

**GAEL XIT BOT V2.4** es un bot de WhatsApp completo y poderoso, desarrollado con Node.js y la biblioteca Baileys. Este proyecto oficial ofrece más de 150 comandos organizados en categorías para administración de grupos, juegos, utilidades, descargas y contenido exclusivo.

### ✨ Destaques

- 🎮 **Juego de la Vieja (Tricky)** con ranking persistente.
- 🤖 **IA Wendel** integrada para conversaciones inteligentes.
- 🛡️ **Sistema AntiLink** con 5 niveles de seguridad.
- 🔞 **Menú Hot** con Waifus desnudas y Hentai (NSFW).
- 👥 **Gestión completa** de grupos (Ban, Promover, Cerrar).
- 🎵 **Descarga de música y videos** de YouTube con un comando.
- 📊 **Sistema de Niveles** y XP para los miembros.
- 📱 **Interface interactiva** con botones y listas.
